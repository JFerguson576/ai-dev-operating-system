# AI Development Operating System

A structured workflow for high-precision AI-assisted software development.

## Principles
- PRD-first development
- Modular rules architecture
- Reusable commands
- Context reset for execution
- Continuous improvement via bug analysis

## Workflow
1. Fill PRD.md
2. Load rules
3. Plan
4. Export execution.md
5. Reset context
6. Execute
7. Log bugs
8. Improve rules
